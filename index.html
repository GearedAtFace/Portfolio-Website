School Projects

Broken Vessel



        Broken Vessel is a metroid-vania set on a hightech spaceship which crashlanded on a planet with medival technology. It was created as my NHTI capstone project with 6 other classmates over a period of about 4 months. My main job on this project was creating level assets, with nearly all of the core level tiles being created by me other then the hallways. While my main job was the art I helped with most parts of the project such as debugging, balance, custom controller support, weapon mechanics, and more.

Dungeon Mungeon



        Dungeon Mungeon is a procedurally generated dungeon crawler using a key system for progress. It began as a personal project looking into creating generation similar to the Dungeoneering dungeons from Runescape. Eventually the project was merged into a group project as a final for a class where we worked for just under a month on making it into a basic game. The simplified explaination of the progression is that you start with a colored key and can enter doors that match that color. Those doors lead to a miniboss where upon defeating them you obtain a different colored key and repeat this process untill reaching the final boss.

Slime Mine



      Slime Mine is a simple VR survival game with the objective of killing endlessly spawning slimes and depositing their bodies for points. I worked on this over two weeks with one other classmate for my VR class on an Oculus. It features a system where we can turn anything into a weapon you can grab and attack with, whether thats the sword and pickaxe the player is given or the dead slimes you kill and trees that decorate the level.

Personal Projects

Planet Traversal



        Planet traversal is a project of mine where I wrote my own physics system and implemented it into a variation of a Lunar Lander game. Instead of the goal being to simply land on a single planet using limited fuel you start in orbit of a planet and have to travel to other planets where you can land and refuel to again travel to yet another planet. This project is still a work in progress but the goal will either be to reach a home planet or you gain score as you succesfully land on larger or further planets.

Explore The Marsh



        This project was created during a game jam with the theme of arcade. I worked on it over 48 hours with two other friends. The original idea was a game similar to the old Namco arcade game, Bosconian, but with a procedurally generated map. My main job on this project was to create a infinitely generating map using a given set of map tiles we created. I also had to implement a system that loads and unloads these tiles as you progress around the map as storing them all put a lot of stress on the computer.
